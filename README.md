# Reserva Espectáculo 🎬

Sistema web de administración de cine desarrollado con ASP.NET MVC Core y C#.

El proyecto permite la gestión completa de películas, funciones, reservas, clientes, empleados y salas, además de ofrecer funcionalidades de reserva online para clientes.

---

# Tecnologías utilizadas

- C#
- ASP.NET MVC Core
- Entity Framework Core
- SQL Server
- HTML
- CSS
- Bootstrap
- Identity
- Visual Studio 2022

---

# Funcionalidades principales

## Clientes
- Registro e inicio de sesión
- Visualización de cartelera
- Filtro por género
- Reserva online de funciones
- Cancelación de reservas
- Visualización de reservas activas e históricas

## Empleados
- Gestión de películas
- Gestión de funciones
- Gestión de salas
- Gestión de clientes
- Confirmación y cancelación de funciones
- Visualización de reservas
- Balance de recaudación

## Sistema
- Control de acceso por roles
- Validaciones de negocio
- Manejo de disponibilidad de butacas
- Cálculo automático de butacas disponibles
- Restricciones de reservas activas
- Cartelera dinámica

# Cómo ejecutar el proyecto

1. Clonar el repositorio

```bash
git clone https://github.com/JoaquinLiendro/cinema-system-dotnet.git
```

2. Abrir el archivo `.sln` con Visual Studio 2022

3. Restaurar paquetes NuGet

4. Configurar la base de datos

5. Ejecutar migraciones si corresponde

6. Ejecutar el proyecto
